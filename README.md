# Online-Healthcare-Management-System

Online Healthcare Management System 
A responsive web-based UI mockup for a hospital and clinic management system, designed to streamline healthcare workflows for clinics, doctors, staff, and patients.

Features
Unified dashboard linking to modules for Patients, Doctors, Appointments, and Reports.

Each module includes searchable, sortable tables with dummy data.

Modern, mobile-friendly interface styled with TailwindCSS.

Consistent navigation across all pages for a seamless user experience.

Pages
Dashboard: Overview and module navigation.

Patients: List and search patient records.

Doctors: List and search doctor profiles.

Appointments: Manage and review scheduled visits.

Reports: Access mock lab, billing, and discharge reports.

Usage
This codebase is intended for demo and project presentation purposes, showcasing layout and navigation only. It can be extended with backend/database integration and fully dynamic components as needed.
